> A little client side js to auto answer my school work from the site https://quill.org

### 🏠 [Homepage](https://github.com/Element21/quil-autoanswer-hack)

## Install

```sh
yarn install
```

## Usage

```sh
yarn run start
```

For production (building zip for real extension)

```sh
yarn run release
```
## Run tests

```sh
yarn run test
```
